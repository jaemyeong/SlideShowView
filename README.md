# SlideShowView

이미지 슬라이드 뷰

## Requirements

- Xcode 13.2.1
- Swift 5.5

## Installation

### Swift Package Manager

```swift
.package(url: "https://github.com/jaemyeong/SlideShowView.git", .upToNextMajor(from: "0.1.0"))
```

## Usage

```swift
import SlideShowView

let slideShowView = SlideShowView()
```
